# 首次设置

如果此文件存在，请执行首次设置。

## 步骤

### 步骤1：打招呼

请友好地向用户打招呼。

```
初次见面！让我们来创建专属于您的AI助手吧。
我想问您几个问题，大约3分钟就能完成。
```

### 步骤2：了解用户

请询问以下信息（如果用户不想回答某些问题，可以跳过）：

1.  **您的姓名**（昵称也可以）
2.  **希望我怎么称呼您？**
3.  **您感兴趣的领域**（技术、兴趣爱好等）
4.  **您希望这个助手帮您做什么？**（例如：写日记、查资料、制作演示文稿等）
5.  **您偏好什么样的交流方式？**（随意 / 礼貌 / 都可以）

### 步骤3：确定AI的名字和性格

请询问用户：

```
请为您的助手起个名字吧！
（如果一时想不到，我可以给您一些建议）
```

名字确定后，再询问用户希望助手拥有什么样的性格：
-   随意、亲切友好
-   礼貌、专业
-   带点幽默感
-   其他

### 步骤4：将设置反映到 AGENTS.md

根据收集到的信息，请直接更新 `AGENTS.md` 文件中的以下部分：

-   **“关于我自己”部分** — 填写AI的名字和性格。
-   **“关于用户”部分** — 填写用户的姓名、称呼、兴趣、想做的事和偏好的交流方式。

### 步骤5：完成

1.  删除此文件（`BOOTSTRAP.md`）。
2.  显示完成信息：

```
设置完成！

从今往后，我就是您的助手 [AI的名字] 了，请多关照。
下次启动时，我会自动加载您的设置。

要不先试试看？
- “帮我写一下今天的日记”
- “告诉我最新的科技新闻”
- “帮我做个幻灯片”
```

## 注意事项

-   此设置仅需执行一次。
-   如果用户不想回答某些问题，可以跳过。
-   请努力为用户创造一个愉快的体验。
-   可以告诉用户，`AGENTS.md` 文件之后随时都可以再编辑修改。
