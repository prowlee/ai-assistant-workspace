## 记忆（记录）

每个会话都是从空白状态启动。以下文件确保了连续性：
- **每日记录：** `memory/YYYYMMDD.md`（如果 `memory/` 目录不存在，则创建它）

记录重要的事情：判断、上下文、需要记住的事项。

### 务必把记忆写进文件

- **记忆是有限的** — 想记住的事情，就写进文件里。
- “脑内笔记”会在会话重启后消失。文件会保留。
- 当用户说“记住这个”、“记录一下”时 → 更新 `memory/YYYYMMDD.md` 文件。
- 不要没写进文件就说“已记录”（务必先操作文件，再报告）。
- **文件 > 记忆**

## 路径设置

| 变量 | 路径 | 说明 |
| :--- | :--- | :--- |
| `[WORKSPACE]` | （仓库根目录） | 工作区根目录（禁止硬编码） |
| `[STATE_DIR]` | （AI 工具的配置目录） | 存放认证信息、媒体文件、密钥等（例如: `~/.xangi/`） |
| `[NOTES_DIR]` | `./notes/` | 笔记保存位置（相对于仓库根目录的路径） |
| `[SKILLS_DIR]` | `./skills/` | 技能存放目录 |
| `[SKILL_DIR]` | （当前执行技能的目录） | 在每个 `SKILL.md` 内部指向其自身所在目录 |

当各技能需要保存笔记时，请使用 `[NOTES_DIR]` 路径。
